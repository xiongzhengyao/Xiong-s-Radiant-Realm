## Tline & ROT & Linecount
$ VTPixelClock = frameRate * lineLengthPixelClock * frameLengthLines $

$ lineReadOutTime(Tline) = 1 / (frameRate * frameLengthLines) $

$ ReadOutTime = Tline * frame\_height $

$ Linecont = Exptime / Tline $

如何理解 linecount，其存在的意义

**<font style="color:rgb(0, 0, 0) !important;">linecount 是曝光时间的核心换算桥梁，</font>**<font style="color:rgba(0, 0, 0, 0.85);">是与曝光时间强绑定的关键参数，指传感器完成曝光所对应的行数。</font>

<font style="color:rgba(0, 0, 0, 0.85);">意义：</font>**<font style="color:rgb(0, 0, 0) !important;">简化曝光时间的量化配置，适配硬件设计逻。Sensor 的曝光时间若直接用时间单位（如毫秒、秒）配置，需复杂的时钟换算电路，而 linecount 以 “行数” 为单位，完美适配 Sensor 逐行曝光的硬件特性。Sensor 的单行处理时间（tline）可通过像素时钟和行长度精准计算，结合曝光时间=linecount×tline 的公式，能轻松将抽象的曝光需求转化为硬件可识别的整数行数。这种方式无需复杂硬件，仅通过简单计数器就能实现曝光时间控制，既降低了 Sensor 的电路设计成本，也避免了时间单位换算带来的精度误差。比如高通平台就通过该逻辑，将曝光需求转化为 linecount 数值写入传感器寄存器。</font>**

# sensor尺寸、35mm等效焦距、FOV、ZOOM倍数的换算关系
EFL（Effective Focal Length）指的是**<font style="color:#DF2A3F;">有效焦距</font>**，它表示镜头中心到焦点的距离。对于厚透镜或者是多片透镜组成的系统（例如照相机或望远镜的镜头），焦距通常以有效焦距EFL来表示。EFL用来描述光学系统聚集光线的能力，并且常用来计算放大系数。镜头的焦距分为像方焦距和物方焦距，像方焦距是指像方主面（后主面）到像方焦点（后焦点）的距离，而物方焦距是指物方主面（前主面）到物方焦点（前焦点）的距离。有效焦距对于确定镜头的视场角和光圈大小等光学特性至关重要

## 等效焦距计算方法详解
### 等效焦距的概念和计算:
    - 等效焦距是将不同尺寸传感器的镜头换算到35mm全画幅上的焦距值
    - 计算公式: **<font style="color:#DF2A3F;">等效焦距 = 实际焦距 × 画幅系数(crop factor)</font>**
    - 目的是统一参照标准,方便比较不同相机的视角
1. <font style="color:#DF2A3F;">画幅系数(crop factor)</font>计算详解:
    - 画幅系数 = 35mm全画幅对角线长度 ÷ 实际传感器对角线长度
    - 35mm全画幅尺寸为36mm×24mm,对角线长度为43.3mm
        * √36<sup>2</sup>+24<sup>2</sup>=43.2666mm
    - 画幅系数越大,表示传感器尺寸相对全画幅越小
2. 传感器对角线长度的精确计算:
    - 使用勾股定理计算: 对角线长度 = √(传感器宽度² + 传感器高度²)
    - 需要先将像素尺寸转换为物理尺寸(mm)
    - 转换比例根据传感器物理大小确定

### 具体计算示例:
1. 示例 1
    - 传感器参数:
        * 分辨率: 4096×3072像素
        * 物理尺寸: 约1/2英寸(7.9mm×5.9mm)
        * 实际焦距: 5.6mm
    - 详细计算步骤:
        * 计算传感器对角线: √(7.9mm² + 5.9mm²) = 7.9mm
        * 计算画幅系数: 43.3mm ÷ 7.9mm = 5.5
        * 计算等效焦距: 5.6mm × 5.5 = 31mm
2. 示例 2 （OV50Q）
    - 传感器参数
        * 分辨率 8192*6144
        * pixel size: 1.197 µm x 1.197 µm
        * 物理尺寸: 约1/1.3" 英寸(9805.824µm×7354.368 µm )
            + 手册是image area: 9824.976 µm x 7392.672 µm ，怀疑是有 offset ,wide 左右 8 pixel， height 16 pixel
        * 实际焦距(EFL)：6.62mm
    - 详细计算步骤:
        * 计算传感器对角线： √(9805.824<sup>2</sup>µm² + 7354.368<sup>2</sup>µm²) = 12.25728mm
        * 计算画幅系数(cropFactor): 43.2666/12.25728=3.53
        * 计算等效焦距: 6.62*3.53=23.3686
